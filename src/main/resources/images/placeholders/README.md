# Vault-Tec Placeholder Images

This directory contains placeholder images for the Mini Game Hub.

## Expected Files

### default-placeholder.png
- **Size:** 200x120 pixels
- **Purpose:** Default placeholder for all minigame cards when specific image is not found
- **Suggested Content:** Vault-Tec logo or blue/yellow gradient with "VT" text

### pip-hack-placeholder.png
- **Size:** 200x120 pixels
- **Purpose:** Placeholder for "Pip-Boy Terminal Hack" minigame
- **Suggested Content:** Green terminal screen with brackets `[ ]` or RobCo terminal aesthetic

### nuka-rush-placeholder.png
- **Size:** 200x120 pixels
- **Purpose:** Placeholder for "Nuka-Cola Rush" minigame
- **Suggested Content:** Colorful Nuka-Cola bottle or puzzle pieces in red/blue colors

## Vault-Tec Color Scheme

- **Primary Blue:** #3C8DBC
- **Accent Yellow:** #FFE600
- **Background Dark:** #1A1A1A
- **Text Light:** #CCCCCC
