# JavaFX MiniGame Hub

Petit hub de mini-jeux développé en JavaFX.  
Le but du projet est de regrouper plusieurs mini-jeux dans une seule application desktop, avec une interface simple et rapide à lancer.

---

## ✨ Fonctionnalités

- Menu principal pour naviguer entre les mini-jeux
- Interface graphique JavaFX
- Architecture modulaire (ajout de jeux possible)
- Projet orienté apprentissage / pratique Java

---

## 🧱 Stack technique

- **Java** (JDK 17+ recommandé)
- **JavaFX**
- **Maven** ou **Gradle** (selon la version du projet)

---
